IPSians Diary was brought up as a vision, which aims to be a one stop shop for the IPS Academy irrespective course.

It aims to create an all in one helpful platform for all IPS Academy students.

The app uses Material UI for an attractive and appealing interface which attracts user to stay on the app for more time.

The app contains many useful features which will help a student in their journey through college. More such features are in development.

We planning to make this app open source and can be found on GitHub.
